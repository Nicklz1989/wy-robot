## 🔧 环境配置

### 本地开发
```bash
npm install
npm run dev
# 访问 http://localhost:3000
```

### 环境变量 (.env.local)
```bash
NEXT_PUBLIC_SUPABASE_URL=https://xxx.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJxxx
SUPABASE_SERVICE_ROLE_KEY=eyJxxx
KIMI_KEY=sk-xxx
ADMIN_USERNAME=Nicklz
ADMIN_PASSWORD_HASH=$2a$10$xxx
```
