# Vercel 密码保护功能配置教程

## 一、功能说明

Vercel 密码保护功能是在部署层面添加的额外安全保护，无需修改代码即可实现。启用后，访问部署的网站时会先弹出 Vercel 的密码验证框。

## 二、启用密码保护的前提条件

1. Vercel Pro 或 Enterprise 套餐（付费套餐）
2. 或者使用 Vercel 部署的专业功能

> **注意**: 免费套餐（Hobby）不支持密码保护功能。如果使用的是免费套餐，可以考虑：
> - 升级到 Pro 套餐（约 $20/月）
> - 或者使用自定义域名 + 第三方密码保护服务

## 三、详细配置步骤

### 步骤 1：进入项目设置

1. 打开 Vercel Dashboard：https://vercel.com/dashboard
2. 选择你的项目 **wy-robots**
3. 点击 **"Settings"**（设置）

### 步骤 2：找到密码保护选项

在 Settings 页面左侧菜单中，向下滚动找到 **"Security"**（安全）选项

### 步骤 3：启用密码保护

1. 点击 **"Security"**
2. 找到 **"Password Protection"**（密码保护）
3. 点击 **"Add Password"** 或开关按钮
4. 输入你想要的密码：
   - 建议密码：`Wangyue2024!`
   - 确认密码

### 步骤 4：配置密码规则（可选）

可以设置：
- 密码复杂度要求
- 是否允许多个密码

### 步骤 5：保存设置

点击 **"Save Changes"** 保存

## 五、访问权限配置

### 整体保护（推荐）

启用密码保护后，整个网站（包括用户端和管理后台）都需要密码访问。

### 路径级别保护（高级）

如果只想保护管理后台，可以配置特定的路径：

1. 在 Password Protection 设置中
2. 找到 **"Protection Bypass"** 或 **"Exceptions"**
3. 添加不需要密码的路径：
   - `/` - 首页
   - `/api/chat` - 对话 API
4. 保留需要保护的路径：
   - `/admin*` - 管理后台

2. 再输入应用内的账号密码

## 八、常见问题解答

### Q1: 免费套餐可以用吗？
A: 不能。Vercel 密码保护需要 Pro 或 Enterprise 套餐。

### Q2: 忘记密码怎么办？
A: 在 Vercel Settings → Security 中重置密码。

### Q3: 可以设置多个密码吗？
A: 可以。在 Password Protection 设置中可以添加多个密码，方便多人使用。

### Q4: 密码保护会影响 SEO 吗？
A: 不会。搜索引擎爬虫可以绕过密码保护，内容仍会被索引。

### Q5: 可以只保护管理后台吗？
A: 可以通过配置路径例外来实现，但配置较复杂。建议使用整体保护。
