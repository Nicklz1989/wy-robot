# 往约智能问答系统 - 编码实施计划

## 一、项目概述

基于使用指南，本项目是一个 **AI 智能问答系统**，采用 Serverless 架构部署于 Vercel 平台，后端服务使用 Supabase。

### 技术栈
| 层级 | 技术 |
|-----|------|
| 前端框架 | Next.js 14 (App Router) |
| 部署平台 | Vercel |
| 后端服务 | Supabase |
| 数据库 | PostgreSQL |
| AI 引擎 | Moonshot AI (Kimi) |
| 限流服务 | Upstash Redis + Ratelimit |
| 样式方案 | CSS Modules / Tailwind CSS |

## 四、开发流程规则

1. **阶段确认**: 每个阶段开始前需提交开发计划，获得确认后启动
2. **编码规范**: 遵循 ESLint + Prettier 规范
3. **测试要求**: 每个模块完成后必须进行单元测试
4. **文档同步**: 边开发边整理技术文档
5. **阶段报告**: 完成后提交完成报告和测试结果
